# 🍄 觸控瑪莉歐冒險 - 手機遊戲

一個專為手機設計的瑪莉歐風格觸控遊戲，支援單擊觸控操作，擁有完整的遊戲體驗和現代化UI設計。

## 🎮 遊戲特色

### 觸控操作
- **單擊跳躍** - 點擊螢幕任意位置讓瑪莉歐跳躍
- **滑動移動** - 向左/向右滑動控制瑪莉歐移動
- **雙擊技能** - 雙擊觸發特殊跳躍
- **長按輔助** - 長按啟用輔助模式

### 遊戲內容
- 🧑‍🍄 **經典瑪莉歐角色** - 完整的動畫和物理效果
- 🪙 **收集金幣** - 觸碰金幣增加分數和獎勵
- 🍄 **蘑菇道具** - 恢復生命和特殊能力
- 👾 **敵人系統** - 避開或踩踏敵人
- 🏗️ **平台跳躍** - 多樣化的平台佈局
- ⚡ **粒子效果** - 豐富的視覺特效

### 技術特色
- 📱 **純手機優化** - 專為觸控設計的UI和控制
- 🎨 **現代化界面** - 漸變色彩和流暢動畫
- 🔊 **音效系統** - 完整的遊戲音效支持
- 💾 **進度保存** - 自動保存遊戲設置和統計
- 📐 **響應式設計** - 適配各種屏幕尺寸
- ⚡ **高性能** - 流暢的60FPS遊戲體驗

## 🕹️ 操作說明

### 觸控操作
- **單擊任意位置** - 瑪莉歐跳躍
- **向左滑動** - 瑪莉歐向左移動
- **向右滑動** - 瑪莉歐向右移動
- **雙擊** - 觸發雙跳技能
- **長按** - 啟用輔助模式

### 鍵盤控制（桌面）
- **空格鍵/↑鍵/W** - 跳躍
- **←鍵/A** - 向左移動
- **→鍵/D** - 向右移動
- **ESC鍵/P** - 暫停/恢復

- 🎯 收集金幣獲得高分

- 🏆 創造個人最佳記錄

## 🚀 部署說明

### 本地運行
```bash
# 直接在瀏覽器中打開 index.html
# 或使用簡單的HTTP服務器
python -m http.server 8000
# 訪問 http://localhost:8000
```

### GitHub Pages部署
1. 將專案上傳到GitHub倉庫
2. 在Repository Settings中啟用Pages
3. 選擇main分支作為源
4. 訪問生成的URL即可遊玩

### 自定義部署
- 修改 `index.html` 中的標題和描述
- 調整 `css/style.css` 中的顏色和佈局
- 添加更多音效文件到 `assets/` 目錄

## 🎨 自定義選項

### 修改遊戲參數
在 `js/mario.js` 中調整：
```javascript
// 跳躍力度
this.jumpPower = -15;

// 移動速度
this.moveSpeed = 5;

// 重力
this.gravity = 0.8;
```

### 添加新道具
在 `js/game.js` 中擴展：
```javascript
// 新的道具類型
case 'star':
    // 星星道具效果
    break;
```

### 自定義UI風格
在 `css/style.css` 中修改：
```css
/* 主色調 */
.game-header {
    background: linear-gradient(90deg, #新顏色1, #新顏色2);
}
```
